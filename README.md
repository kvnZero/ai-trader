## Trader

Starter project using:

- Python 3.12
- Flask
- Tailwind CSS
- uv

### Development

Install Python dependencies:

```bash
uv sync
```

Install frontend dependencies:

```bash
npm install
```

Run the Flask app:

```bash
uv run flask --app main:app --debug run
```

Watch Tailwind CSS:

```bash
npm run dev
```

Create a production CSS build:

```bash
npm run build
```
